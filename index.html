Tags básicas do HTML

Voltar

As tags HTML básicas são essenciais para definir a estrutura de uma página web, organizando elementos como títulos, parágrafos e quebras de linha. Além disso, permitem a formatação do texto, possibilitando a aplicação de estilos como negrito, itálico e sublinhado.
Existem mais de 100 tags HTML, porém, as mais básicas são:

  • <html>: Define o inicio do documento HTML.
  • <head>: Define o cabeçalho do documento, onde ficam os metadados.
  • <body>: Define o corpo do documento, onde fica o conteúdo principal.
  • <footer>: Define o rodapé do documento/site.

Atributos das Tags

Voltar

São informações que definem como uma tag HTML se comporta. Eles são usados para modificar o comportamento da tag ou fornecer metadados. Eles são inseridos dentro da abertura da tag e seguem o formato nome="valor".
Principais atributos em HTML:

  • id: identifica um elemento de forma única.

    <p> id="intro">Texto de introdução.</p>

  • class: Agrupa elementos com o mesmo estilo.

    <div> class="container">Conteúdo aqui</div>

  • src: Define a origem de imagens ou scripts.

    <img> src="imagem.jpg"> alt="Descrição da imagem">

  • href: Especifica o destino de um link.

    <a href>="https://exemplo.com">Clique aqui</a>

    • Há também as tags alt: que define um texto alternativo para imagens. style: adiciona estilos CSS diretamente na tag. target: controla como os links abrem (_blank para nova aba).

Textos e Listas

Voltar

Em HTML, textos e listas* são estruturados utilizando tags específicas para organização e formatação do conteúdo. Os textos podem ser estilizados e organizados com tags como <p> para parágrafos e <h1> até <h6> para títulos, que ajudam na hierarquia e legibilidade do conteúdo.
As listas* servem para organizar itens de forma ordenada ou não ordenada. As listas ordenadas seguem uma sequência numérica ou alfabética, enquanto as não ordenadas utilizam marcadores.

  1. <ol> é uma lista ordenada numérica (pai) onde não é possível escrever textos, elas necessitam da tag <li>(filha) para funcionar.
  • <ul> é uma lista não ordenada, onde os itens são exibidos com marcadores, como pontos.

Usando a tag <li>, é possível inserir uma lista dentro da outra, podendo usar tanto a ol quanto a ul. Esses elementos são essenciais para tornar o conteúdo mais estruturado, acessível e legível em uma página web.
