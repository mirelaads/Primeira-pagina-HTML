Tags básicas do HTML

Voltar

As tags HTML básicas são essenciais para definir a estrutura de uma página web, organizando elementos como títulos, parágrafos e quebras de linha. Além disso, permitem a formatação do texto, possibilitando a aplicação de estilos como negrito, itálico e sublinhado.
Existem mais de 100 tags HTML, porém, as mais básicas são:

  • <html>: Define o inicio do documento HTML.
  • <head>: Define o cabeçalho do documento, onde ficam os metadados.
  • <body>: Define o corpo do documento, onde fica o conteúdo principal.
  • <footer>: Define o rodapé do documento/site.
